# Local AI Knowledge Space Frontend

Next.js 14 App Router frontend for uploading, searching, chatting with, and graphing a local AI knowledge base.

## Setup

```bash
cd frontend
bun install
cp .env.example .env.local
bun run dev
```

Frontend runs at `http://localhost:3000`.

## Commands

```bash
bun run dev
bun run build
bun run typecheck
```

No Tailwind or shadcn/ui. Styling uses CSS Modules plus `app/globals.css`.
